# DRL-LEO-Beam-Positioning
The Python code for DRL-Based Beam Positioning for LEO Satellite Constellations with Weighted Least Squares
  * PPO
  * DDQN
  * DQN
  * ALG-B (Beam-Based Positioning) [1]
  * LSTM-WLS [2]
  * Proposed DQN-WLS

[1] M. Elsanhoury, J. Koljonen, M. Elmusrati and H. Kuusniemi, "A Novel Beam-Based Positioning Paradigm Via Opportunistic Signal of Future Massive MIMO LEO Satellite Constellations," 2024 International Conference on Localization and GNSS (ICL-GNSS), Antwerp, Belgium, 2024

[2] I. Sbeity, C. Villien, C. Combettes, B. Denis, E. V. Belmega and M. Chafii, "A Novel Satellite Selection Algorithm Using LSTM Neural Networks For Single-epoch Localization," 2023 IEEE/ION Position, Location and Navigation Symposium (PLANS), Monterey, CA, USA, 2023, pp. 105-112
